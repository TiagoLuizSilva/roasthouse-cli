# 🍖 RoastHouse

Sistema simples de gerenciamento para uma casa de assados, desenvolvido em Python (CLI).

## 📌 Sobre o projeto

O **RoastHouse** foi criado com base em um problema real observado em um açougue de bairro, onde o controle de vendas era feito manualmente em um caderno, dificultando a organização em momentos de alto movimento.

Este projeto tem como objetivo oferecer uma solução simples, prática e funcional para controle de produtos e vendas.

---

## ⚙️ Funcionalidades

- 📋 Cadastro de produtos
- 📦 Controle de estoque
- 🗑️ Remoção de produtos
- 💰 Simulação de vendas
- 🔄 Atualização automática do estoque
- 🎨 Interface colorida no terminal

---

## 🧠 Tecnologias utilizadas

- Python 3
- Manipulação de arquivos `.txt`
- Estrutura modular (separação de responsabilidades)

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/TiagoLuizSilva/roasthouse-cli.git
```


2. Acesse a pasta: 

```bash
cd RoastHouse
```


3. Execute o programa:

```bash
python main.py
```
